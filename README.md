# [Simple Shell Project](https://github.com/Themby2000/simple_shell.git)
***

## ABOUT
***
This project tasks to create a simple UNIX command interpreter. project is a simple version of the linux shell tasked by ALx SE in the `ALx Low-level programming projects`
It is created using the C programming Language and it can do many functionalities that a real shell does. To know more about [SHELL](https://www.geeksforgeeks.org/introduction-linux-shell-shell-scripting/) and how to write your very own [Linux shell](https://www.geeksforgeeks.org/making-linux-shell-c/).
***

## INSTRUCTIONS
***
> Step 1: Clone repository using this command
```
git clone https://github.com/Themby2000/simple_shell.git
```
> Step 2: change dir to simple_shell
> Step 3: Compile the C files this way:
```
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh
```
> Step 4: Run the shell ./hsh
> important EXITING the shell:
>> 1: Type the command "exit"
```
Exit
```
>> 2: or do Ctrl + D
